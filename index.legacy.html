User Guide

1. Designing Walls & Rooms

Use the Wall tool to draw custom wall segments. Use the Room tool for quick rectangular enclosures. Walls will automatically snap to each other when "Sticky" mode is enabled.

2. Adding Elements

Expand the Elements section in the sidebar to add doors, windows, and furniture. Once placed, use the Select tool to move or resize them.

3. Precision Tools

Enable Grid or Vastu from the View menu for better alignment. Use the Guide tool to show alignment crosshairs for pixel-perfect placement.

4. Exporting

When your design is complete, use File > Export as PNG to save a high-resolution top-down image of your floor plan.
